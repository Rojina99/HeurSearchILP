## Dataset used for testing generated heuristic
`saved_test_data_45` 
    data used for getting heursitic without a star, here the name 45 comes due to use of atmost 45 states of each cost to go or depth 
`saved test_data_astar` 
    data used to solve the states with a star
    can use eight_puzzle_states.pkl to compare true cost to go

`saved_test_data_45` will be generated if passed as `--test_dir` argument to `test_heur_script/test_heuristic_performance_no_a_star.py`

`saved test_data_astar` will not be generated automatically, create script to generate
